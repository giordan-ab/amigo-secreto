# 🎁 Amigo Secreto  

Este es un proyecto simple para organizar un sorteo de **Amigo Secreto** 🎉.  

## 🚀 Funcionalidades  
- ➕ Añadir nombres de amigos  
- 🎲 Sortear un amigo al azar  
- 📝 Mostrar el resultado en pantalla
